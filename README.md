# Store App

## Requirements

Before getting started, make sure you have the following installed:

- **Node.js**: version `22.13.0`
- **PostgreSQL**

## Installation

1. Clone the repository:

```bash
git clone git@github.com:ravnhq/BE-Nerdery-Week5-Challenge.git
```

2. Install dependencies

```bash
npm i
```

## Running the app

```bash
npm run dev
```
